# watering-sistem-using-makecode

Features:
Automatic Watering: Turns pump on when soil gets too dry

Adjustable Threshold: Use buttons A/B to change sensitivity

Visual Feedback: Icons show soil moisture status

Manual Check: Shake to see current moisture percentage

Safety: Prevents over-watering with timed pump operation

How to Adjust:
Button A: Lowers threshold (waters more frequently)

Button B: Raises threshold (waters less frequently)

Default: Threshold = 400 (40% on display)

Customization Tips:
Change wateringTime to adjust how long the pump runs

Modify dryThreshold range (200-800) for different soil types

Adjust the mapping values if your sensor reads differently

The system will automatically maintain your plants' moisture levels and you can fine-tune the sensitivity based on your specific plants' needs!
